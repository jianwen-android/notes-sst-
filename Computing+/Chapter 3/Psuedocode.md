# Syntax

## Input/Output

### Input

```
INPUT variable
```

### Output

```
OUTPUT variable
```

## Selection Constructs

```
IF condition THEN
	action
ELSE IF condition THEN
	action
ELSE
	action
ENDIF
```

## Iteration Constructs

### For loops

```
FOR count 1 to n
    action
NEXT variable //to increment count
```

### While loops

```
WHILE condition //is true
	action
ENDWHILE
```
